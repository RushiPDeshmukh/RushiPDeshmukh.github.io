    Visual Odometry on Omnidirectional Mobile Robot

    Designed a real time system with hardware omnidirectional robot with Raspberri Pi and Oak D Pro camera to calculate the robot odometry using camera frames sent as compressed images to host machine. Implemented a real time node which can work with datasets and continuous camera stream to get visual odometry using Perspective-N-Points and RANSAC on SIFT feature matches between consequtive images.
