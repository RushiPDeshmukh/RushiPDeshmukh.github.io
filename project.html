    Lanelet Mapping using Semantic Segmentation and Depth Estimation

    Developed a robust lane detection algorithm using Semantic Segmentation for extracting drivable surfaces along with edge detectors and line estimators. Edges extracted using Canny edge detector were passed to a Hough line estimator and filtered using slope heuristics to get lane markings in the dataset. Designed a ROS2 architecture to map a City Grid into Lanelet maps for Behavioral planning.

    Semantic Segmentation, Depth Maps, Canny Edge Detector, Hough Line Estimator
    Successfully extracted driving surfaces and detected lanes for an autonomous vehicle to allow navigation and behavior planning. Lane detection was done using classical computer vision techniques allowing for low computational complexity as compared to Deep learning apporaches.
